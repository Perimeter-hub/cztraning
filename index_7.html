🔊 Настройка голоса
Для правильного произношения чешских слов нужен чешский голос на вашем устройстве.
✅ Чешский голос найден!
Голос: —
Произношение будет правильным. Приятного обучения!
⚠️ Чешский голос не найден
Без чешского голоса слова могут произноситься неправильно (например, byt как «байт» вместо «бит»).
📱 Android — установка чешского голоса
1. Настройки → Общие управление
2. Язык и ввод → Синтез речи
3. Нажмите ⚙️ у Google Text-to-Speech
4. Язык → добавить Čeština (Česká republika)
5. Перезапустите приложение
🍎 iPhone / iPad — установка чешского голоса
1. Настройки → Универсальный доступ
2. Дикторский текст → Голоса
3. Найдите Чешский → выберите голос Zuzana
4. Скачайте и перезапустите браузер
💻 Windows — установка чешского голоса
1. Пуск → Параметры → Время и язык
2. Язык → Добавить язык → Čeština
3. При установке отметьте «Синтез речи»
4. Перезапустите Chrome
🍎 Mac — установка чешского голоса
1. Apple меню → Системные настройки
2. Универсальный доступ → Дикторский текст
3. Системный голос → Настроить → Zuzana (Czech)
4. Скачайте голос, перезапустите Chrome
